Afritec Bridge LMS API

Welcome to the Afritec Bridge Learning Management System API. This is the backend service that powers the Afritec Bridge LMS platform.

API Endpoints

The API provides endpoints for:

  • User authentication and management
  • Course content and structure
  • Student enrollments
  • Quizzes and assignments
  • Opportunities

API Documentation

For detailed API documentation, please refer to the API documentation provided by the Afritec Bridge team.

Status
